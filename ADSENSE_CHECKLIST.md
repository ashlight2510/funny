## 📋 AdSense 신청 전 확인사항

### 콘텐츠 품질
- [ ] 각 서브 서비스가 실제로 작동하는지 확인
- [ ] 콘텐츠가 충분히 많고 유용한지 확인 (최소 20-30개 페이지 권장)
- [ ] 저작권 침해 없는 원본 콘텐츠인지 확인

### 트래픽
- [ ] 일일 방문자 수집 (최소 100명 이상 권장)
- [ ] 체류 시간 확보 (사용자가 여러 서비스를 이용하도록)
- [ ] 자연스러운 트래픽 (검색엔진, 소셜미디어 등)

### 사이트 구조
- [ ] 모든 링크가 정상 작동하는지 확인
- [ ] 404 에러 페이지 없음
- [ ] 로딩 속도 최적화 (3초 이내 권장)

### 법적 요구사항
- [ ] 개인정보처리방침에 실제 연락처 정보 포함
- [ ] 이용약관이 서비스에 맞게 작성되었는지 확인
- [ ] 광고 정책 준수 (성인 콘텐츠, 폭력 등 금지)

## 🚀 AdSense 신청 단계

### 1단계: 사이트 준비
1. 모든 서브 서비스가 정상 작동하는지 확인
2. 개인정보처리방침과 이용약관 검토
3. 사이트맵을 Google Search Console에 제출

### 2단계: Google Search Console 등록
1. https://search.google.com/search-console 접속
2. `funnyfunny.cloud` 도메인 추가
3. DNS 인증 또는 HTML 파일 업로드로 소유권 확인
4. 사이트맵 제출 (`sitemap.xml`)

### 3단계: Google Analytics 확인
1. Google Tag Manager에서 Analytics 태그가 정상 작동하는지 확인
2. 최소 1주일 이상 데이터 수집

### 4단계: AdSense 신청
1. https://www.google.com/adsense 접속
2. 사이트 URL 입력: `funnyfunny.cloud`
3. 사이트 정보 입력
4. 개인정보처리방침 및 이용약관 링크 제공
5. 신청 제출

### 5단계: 승인 대기
- 일반적으로 1-14일 소요
- 이메일로 승인/거부 알림 수신
- 거부 시 피드백 확인 후 수정 후 재신청
